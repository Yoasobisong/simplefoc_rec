# AS5600 磁编码器读数测试

## 简介
读取 AS5600 磁编码器的角度数据，并在 SSD1306 OLED 上实时显示，同时通过串口输出详细数据。用于验证：
- AS5600 I2C 通信是否正常
- 磁铁安装位置是否合适
- 编码器角度读数是否正确

## 硬件需求
| 器件 | 说明 |
|------|------|
| STM32F103C8T6 最小系统板 | 主控 |
| AS5600 磁编码器模块 | 12-bit, I2C 接口 |
| 径向充磁圆形磁铁 | 装在电机轴上 |
| SSD1306 OLED 0.96寸 128x64 | 显示数据 |
| JLink 调试器 | SWD 烧录 |

## 接线

```
STM32F103C8         AS5600
├── PB6 ──────────── SCL    (硬件 I2C1)
├── PB7 ──────────── SDA    (硬件 I2C1)
├── 3.3V ─────────── VCC
└── GND ──────────── GND

STM32F103C8         SSD1306 OLED
├── PB4 ──────────── SCL    (软件 I2C)
├── PB3 ──────────── SDA    (软件 I2C)
├── 3.3V ─────────── VCC
└── GND ──────────── GND

STM32F103C8         USB-TTL (可选，串口调试)
├── PB10 ─────────── RXD    (USART3 TX)
├── PB11 ─────────── TXD    (USART3 RX)
└── GND ──────────── GND
```

## 编译烧录

```bash
pio run               # 编译
pio run -t upload      # 烧录
pio device monitor     # 串口查看 (115200)
```

## 预期现象

### OLED 显示
```
AS5600 Encoder Test
RAW: 2048 / 4095
DEG: 180.0
MAG:OK AGC:128
```

### 串口输出 (115200, Tab 分隔)
```
RAW    ANGLE  DEG    STATUS     AGC    MAG
2048   2050   180.0  OK         128    1500
2100   2102   184.6  OK         128    1502
```

### AS5600 读数说明
| 数据 | 范围 | 说明 |
|------|------|------|
| RAW | 0 ~ 4095 | 原始角度值 (12-bit, 无滤波) |
| DEG | 0.0° ~ 360.0° | 换算后的角度 = RAW × 360 / 4096 |
| STATUS | OK / TOO STRONG / TOO WEAK / NO MAGNET | 磁铁检测状态 |
| AGC | 0 ~ 255 | 自动增益值，反映磁场强度 |
| MAG | 0 ~ 4095 | 磁场幅度原始值 |

### 磁铁状态排查
| 显示 | 含义 | 处理 |
|------|------|------|
| `OK` | 磁场强度正常 | 无需调整 |
| `TOO STRONG` | 磁铁太近或太强 | 增加磁铁与芯片的距离 |
| `TOO WEAK` | 磁铁太远或太弱 | 减小距离或换更强磁铁 |
| `NO MAGNET` | 未检测到磁铁 | 检查磁铁是否安装、方向是否正确 |
